Examples

Tree

Description
A tree is a hierarchical list that allows different branches of the tree to be hidden or shown under user control.
Example
<xbl>
 <binding element="div.tree" extends="http://dist.meekostuff.net/XBLUI/default/UI.xml#tree" />
 <binding element="div.tree li" extends="http://dist.meekostuff.net/XBLUI/default/UI.xml#treeitem" />
</xbl>

Menu

Description
A menu is a tree where only one branch is visible at a time and sub-menus popup adjacent to higher menus.
Example
<xbl>
 <binding element="div.menu" extends="http://dist.meekostuff.net/XBLUI/default/UI.xml#menu" />
 <binding element="div.menu li" extends="http://dist.meekostuff.net/XBLUI/default/UI.xml#treeitem" />
</xbl>

Tabbox

Description
A tabbox is a viewBox (surrounding the content) preceded by a list of skip links styled as a set of tabs.
Example
<xbl>
 <binding element="div.tabpanels" extends="http://dist.meekostuff.net/XBLUI/default/UI.xml#switchBox" />
 <binding element="div.tabs" extends="http://dist.meekostuff.net/XBLUI/default/UI.xml#tree" />
 <binding element="div.tabs li" extends="http://dist.meekostuff.net/XBLUI/default/UI.xml#treeitem" />
</xbl>
